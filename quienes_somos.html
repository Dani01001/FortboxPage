Somos un grupo de estudiantes de 2do año del Bachillerato Técnico en Informática, y desarrollamos un casillero inteligente que se desbloquea mediante tarjetas RFID. Este proyecto nos permite aplicar nuestros conocimientos de electrónica y programación, combinando seguridad y tecnología para crear un sistema eficiente y moderno. El casillero se desbloqueará acercando una tarjeta RFID autorizada, lo que ofrece una solución práctica y segura para el almacenamiento personal. Este tipo de tecnología tiene aplicaciones en múltiples entornos, como instituciones educativas, oficinas y gimnasios.
